How to Use This Book

A guide for parents, teachers, tutors, and learners

For Families: Shared Reading

This dictionary is designed for parents and children to explore together. Each page features a thematic scene from one of the 7 Lands of the Rhythm Quest. Point to each numbered item in the illustration, say the word aloud, then find it in the vocabulary table below.

  1. LOOK Find the numbered items in the scene illustration at the top of each page.
  2. SAY Read the word aloud. Use the phonetic guide (/fuh-NEH-tik/) to help with pronunciation.
  3. CONNECT Read the sentence context to see how the word is used in the Rhythm Quest story.
  4. TRANSLATE Write the word in your home language in the translation column. This makes the book yours!
  5. PLAY Look for the character tips from the hero duos. They offer fun activities to practice each group of words.

For Teachers & Programs

This dictionary is aligned with the thematic scope of the Oxford Picture Dictionary and supports Title III (English Language Acquisition) programming. Each Land maps to standard ESOL survival vocabulary topics. The sub-scenes can be used as standalone lesson units.

💡 Print Referencing Tip

When reading with a child, point to each word as you say it. Tap the number in the illustration, then trace your finger to the matching number in the table. This builds the crucial connection between spoken and written language.
